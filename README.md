# data_science_open_projects

A collection of data science projects showcasing various machine learning algorithms, time series forecasting, and big data analytics.

## Projects

### 1. Naive Bayes Classifier
Location: `naive_bayes_classifier/`

A complete tutorial and implementation of the Naive Bayes classification algorithm from scratch using NumPy. Includes:
- Theoretical background on Bayesian classification
- Step-by-step implementation
- Educational Jupyter notebook

[View Project](./naive_bayes_classifier/)

### 2. Prophet Time Series Forecast
Location: `prophet_time_series-forecast/`

Time series forecasting using Facebook's Prophet library with Google Analytics data.

[View Project](./prophet_time_series-forecast/)

### 3. PySpark Healthcare Data Analytics
Location: `pyspark_project/`

Big data analytics project using PySpark for synthetic healthcare data analysis.

[View Project](./pyspark_project/)

## Getting Started

Each project has its own README with specific instructions. Navigate to the project directory and follow the setup instructions provided.
